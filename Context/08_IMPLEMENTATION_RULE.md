# LevelUp

Implementation Rules

Version 1.0

---

# PURPOSE

This document defines mandatory implementation rules for the AI Agent.

The AI Agent must follow every rule defined below.

These rules take priority over implementation preferences.

The objective is to ensure the final project remains maintainable, scalable, and production-quality.

---

# GENERAL RULES

The AI Agent MUST behave like a Senior Frontend Engineer.

The AI Agent MUST prioritize maintainability.

The AI Agent MUST prioritize readability.

The AI Agent MUST avoid shortcuts.

The AI Agent MUST build reusable code.

The AI Agent MUST follow all previous context documents.

---

# TECHNOLOGY RULES

Allowed Technologies

HTML5

CSS3

Vanilla JavaScript (ES Modules)

Firebase Authentication

Cloud Firestore

Netlify

Do NOT introduce additional frontend frameworks.

Do NOT install unnecessary libraries.

The project should remain lightweight.

---

# SPA RULES

The application MUST use one HTML page only.

Never create:

login.html

dashboard.html

profile.html

quest.html

Every screen must be rendered dynamically.

The browser should never refresh during navigation.
