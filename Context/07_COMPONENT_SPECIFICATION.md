# LevelUp
Component Specification

Version 1.0

---

# PURPOSE

This document defines every reusable UI component used inside LevelUp.

Every page MUST be built using reusable components.

Components should be independent.

Components should never contain business logic.

Business logic belongs inside Services.

Components are responsible only for displaying data and emitting user interactions.

---

# DESIGN PRINCIPLES

Every component should follow the Design System.

Large Border

Rounded Corners

Offset Shadow

Bold Typography

Minimal Decoration

Responsive

Accessible

Reusable
